  • Created a Calendar table.

  • Calculated Transaction Duration by finding the difference between Start Time and End Time.
  • Created a time_of_day column using a SWITCH statement to group the Start Time values into Night, Morning, Afternoon, and Evening categories.
  • Created an Amount table using the DAX UNION function to combine all transaction amount columns across all tables
  • Binned the Amount column in the Amount table into intervals of ‚Ç¶10,000.
  • Created a time_period dimension table to link time periods across all tables.
  • Created a Time Table with an Hour column to synchronize time across all tables.
  • Developed a DAX measure to calculate the average transaction amount.
  • Developed a DAX measure to calculate the overall average transaction duration amount.
  • Developed a DAX measure to calculate the overall transaction amount.
  • Developed a DAX measure to calculate the total number of transactions.
